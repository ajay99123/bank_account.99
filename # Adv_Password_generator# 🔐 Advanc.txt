# Adv_Password_generator# 🔐 Advanced Password Generator (Tkinter GUI)

Hey everyone!  
I built a simple but powerful **password generator** using Python’s Tkinter library. The idea was to make password creation easy, secure, and user-friendly with a graphical interface instead of just command-line scripts.

---

## ✨ What it Does
- Lets you choose the **length** of your password.
- Gives you control over whether to include:
  - Letters (A–Z, a–z)
  - Numbers (0–9)
  - Symbols (!@#$%^&* etc.)
- Makes sure the password is **secure** by including at least one character from each selected type.
- Randomizes everything using Python’s `secrets` module (so it’s cryptographically secure).
- Adds a **Copy to Clipboard** button for quick use.

---

## 🖥️ How to Run
1. Save the script as `password_generator_gui.py`.
2. Run it with Python:
   ```bash
   python password_generator_gui.py
